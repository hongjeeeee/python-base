# 2.4 연산자
파이썬에는 변수나 숫자에 대해서 다양한 작업을 할 수 있는 다음과 같은 연산자가 있다.
- 산술 연산자(arithmetic operators)
- 할당 연산자(assign operators)
- 포함 연산자(membership operators)

### 산술연산자
---
파이썬의 기본 산술연산자는 덧셈, 뺄샘 등과 같은 종류가 있으며 다음 표에서 정리하였다. 예를 들어, 2³와 같은 경우는 기본 함수를 사용하여 `pow(2, 3)`으로 하거나 `2**3`으로 사용할 수 있다. 다음 표에서는 기본적인 연산을 구현하는 방법을 보여주고 있다.

| 연산자 | 내용 | 사용 예 |
| :--- | :--- | :--- |
| + | 덧셈 | x + y |
| - | 뺄셈 | x - y |
| * | 곱셈 | x * y |
| ** | 승셈 | x ** y |
| / | 나눗셈 | x / y |
| % & | 나눗셈 나머지 | 5 % 2 = 1 |
| // | 나눗셈 몫 | 10 // 3 = 3 |

### 할당연산자
---
프로그램에서 값들을 변수에 할당하는 할당연산자가 있으며 다음 표에서 정리하여 보였다.

| 연산자 | 내용 | 사용 예 |
| :--- | :--- | :--- |
| = | 3을 x에 할당 | x = 3 |
| += | x에 2를 더하고 그 결과를 다시 x에 할당 | x += 2 |
| -= | x에서 1을 빼고 그 결과를 다시 x에 할당 | x -= 1 |
| *= | x에 4를 곱하고 그 결과를 다시 x에 할당 | x *= 4 |
| /= | x를 2로 나누고 그 결과를 다시 x에 할당 | x /= 2 |

할당 연산자에 대한 사용 예를 다음 프로그램에서 보여주고 있다.
```python
a = 3
print(a)
a += 2
print(a)
a /= 3
print(a)
```
- `a = 3`에서 숫자 `3`을 변수 `a`에 할당하고 있으며 이제 `a`는 `3`의 값을 갖는다.
- `a += 2`는 `a = a + 2`와 같으며 오른쪽의 연산을 수행한 후 왼쪽으로 할당하는 과정이다.
- 사용 예에서는 `a + 2 = 3 + 2 = 5`이므로 현재 `a`의 값은 `5`이다.
- 다른 할당 연산자에 대해서도 같은 방식으로 생각할 수 있다.

이 프로그램을 실행하면 다음과 같은 결과를 얻는다.
```python
3
5
1.6666666666666667
```
독자들은 왜 이러한 결과가 나왔는지 꼼꼼하게 자세히 살펴보면 다른 사람들이 작성한 프로그램을 볼 때 더 이해가 쉬울 것이다.

### 포함연산자
---
포함연산자의 기능은 연산자의 오른쪽에 있는 항목에 왼쪽의 항목이 존재하는가를 묻는 것이며 다음과 같이 사용한다. 예를 들어, 문자열 `text = '3times'`에 `'3'`이 있는가를 확인하기 위해서는 다음 표에서와 같이 포함연산자를 적용할 수 있다. 여기서 `'3'`은 숫자가 아니라 문자열의 한 요소로 되어 있으므로 따옴표를 사용해야 함을 유의한다.

| 연산자 | 내용 | 사용 예 |
| :--- | :--- | :--- |
| in | 포함된 내용을 확인 | '3' in text |
| not in | 포함되지 않은 내용을 확인 | '3' not in text |

다음 프로그램은 문자열 `text = 'Python is fun'`에 `'fun'`이라는 문자열이 존재하는가를 판정해주는 것이다.
```python 
text = 'Python is fun'
check1 = 'fun' in text
print(check1)
check2 = 'boring' in text
print(check2)
```
이 프로그램의 실행 결과는 각각 다음과 같다.
```python
True
False
```
이 프로그램에서 `check1 = 'fun' in text`에서는 참이 되므로 결괏값이 `True`가 나오게 되며, 반면에 `check2 = "boring" in text`은 `text`에 `'boring'`이 없으므로 거짓이 되므로 `Fasle`라는 결과를 얻는다.
