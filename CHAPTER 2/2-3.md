# 2.3 문자/문자열
문자나 문자열은 따옴표로 둘러싸인 데이터를 의미한다. 홑 따옴표 `'` 또는 곁 따옴표 `"`를 사용할 수 있으며, 3중 따옴표 `"""`는 설명(comments)을 이어갈 때 사용한다. 다음의 프로그램 예에서 볼 수 있듯이 문자나, 문자열 그리고 공백을 포함한 문자열 모두 동일하게 문자열로 인식됨을 알 수 있다.
```python
u = "U"
t = "Truth"
s = "Python is fun."
g = """Programming with Python is a good initiative for anyone who dreams of successful career in the future."""
print(type(u))
print(type(t))
print(type(s))
print(type(g))
```
이 프로그램을 수행하면 다음과 같이 각 변수의 종류를 알 수 있으며, 네 가지 모두 문자열로 인식이 됨을 알 수 있다. 
```python
<class 'str'>
<class 'str'>
<class 'str'>
<class 'str'>
```
