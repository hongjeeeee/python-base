# 2.2 숫자
파이썬에서 사용되는 숫자의 종류는 정수(integer), 실수(float), 복소수(complex) 등이 있다. 다른 프로그램 언어와 달리 사용하기 전에 숫자의 종류를 미리 선언할 필요는 없다.

| 코드 | 형식 | 설명 |
|:---:|:---|:---|
| int | k = 1 | 정수 |
| float | x = 3.14 | 유동 소수점 실수 |
| complex | r = 1 + 4j | 복소수

다음 프로그램은 정수, 실수 및 복소수를 변수 `k`, `x`, `r`에 각각 할당하는 과정을 보인 것이다. 
```python
k = 1 
x = 3.14
r = 1 + 4j
print(type(k))
print(type(x))
print(type(r))
```
데이터 종류에 대한 정보는 `type()` 함수를 사용하면 쉽게 알아볼 수 있다. 위 프로그램에서 사용된 `k`, `x`,  `r`에 대한 데이터 종류는 다음에서 볼 수 있듯이 각각 정수, 실수, 복소수로 규정되어 있다.
```
<class 'int'>
<class 'float>
<class 'complex'>
```

### 데이터 종류의 변환
---
파이썬에서는 연산 중에 필요한 경우에는 자동으로 데이터의 종류를 적절하게 변환해 준다. 다음 예에서는 다른 데이터가 함께 사용되는 연산에서 데이터 종류의 변환이 발생함을 보여주고 있다. 
```python
k = 1
x = 3.14
r = 1 + 4j 
z = k + x
q = k + r
s = x + r
print(z)
print(q)
print(s)
print(type(z))
print(type(q))
print(type(s))
```
- 다음 예에서, 정수 `k`와 실수 `x`의 연산인 `z = k + x` 수행하면 결과는 실수 `<class 'float'>`로 되어 연산 과정에서 정수가 실수로 변환되어 수행되었음을 볼 수 있다.   
  ```python
  k = 1
  x = 3.14
  z = k + x
  print(z)
  print(type(z))
  ```
  이 프로그램을 수행하면 다음과 같다.
  ```python
  4.140000000000001 # 뒤에 1이 붙는 이유는 이진수로 소수를 표현하는 한계 때문에 생기는 부동소수점 오차
  <class 'float'>
  ```

- 이번에는 정수 `k`와 복소수 `r`의 연산 `q = k + r`에서 결과는 `<class 'complex'>`로 나타나 복소수로 변환되었음을 알 수 있다.
  ```python
  k = 1
  r = 1 + 4j
  q = k + r
  print(q)
  print(type(q))
  ```
  이 프로그램을 수행하면 다음과 같다.
  ```python
  (2+4j)
  <class 'complex'>
  ```

- 또한, 실수 `x`와 복소수 `r`의 연산 `s = x + r`에서 결과는 `<class 'complex'>`와 같이 복소수로 변환되었다.
  ```python
  x = 3.14
  r = 1 + 4j
  s = x + r
  print(s)
  print(type(s))
  ```
  이 프로그램을 수행하면 다음과 같다.
  ```python
  (4.140000000000001+4j)
  <class 'complex'>
  ```

- 결론적으로 파이썬에서는 자동으로 연산의 결과를 적절한 데이터의 종류로 변화해 줌을 알 수 있다.
