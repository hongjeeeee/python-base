### 3.2 리스트와 객체의 개념
---
- 파이썬은 객체 지향적인 환경(OOP: Object-Oriented Programming)을 구현하기 때문에 데이터에 대한 작업도 이 방식을 적용한다.
- 객체에 대한 작업은 방법(methods)을 사용하는 것과 함수(functions)를 사용하는 것으로 나뉜다.
- 파이썬에서는 리스트도 객체의 하나로 간주하기 때문에 방법과 함수를 사용하여 필요한 작업을 수행할 수 있다.

### 리스트에서의 방법 사용법
---
리스트에서의 방법은 `객체.방법`과 같은 방식으로 사용된다. 예를 들어, `team.append()`로 사용되며 여기서 `team`은 객체(object)이고 `append()`는 방법(method)이다. 다음은 리스트에서 사용되는 몇 가지의 방법을 정리한 것이다.

| 방법(methods) | 내용 | 예 |
| :--- | :--- | :--- |
| append() | 요소를 끝에 추가 | team.append("and") |
| insert() | 요소를 원하는 위치에 추가 | team.insert(0, "you") |
| reverse() | 역순으로 | team.reverse() |
| remove() | 요소를 제거 | team.remove("and") |
| sort() | 순서대로 | team.sort() |
| count() | 요소의 갯수 | team.count("me") |

다음은 위의 표에서 언급한 방법들을 프로그램으로 작성한 것이다. 먼저, 요소 `and`를 넣어서 리스트 `team`을 만든다. 이번에는 `append()` 방법을 사용하여 `you`를 리스트에 추가하여 결과를 확인한다.
```py
team = ["and"]
team.append("you")
print(team)
```
결과는 다음과 같다.
```py
['and', 'you']
```

리스트 `team`에는 두 개의 요소 `and`와 `you`가 들어가 있다. 리스트에서 원하는 위치에 요소를 추가하고 싶은 경우는 `insert()` 방법을 사용한다. `team.insert(0, "me")`는 리스트의 제일 앞에 `me`를 추가하는 것이다.
```py
team.insert(0, "me")
print(team)
```
결과는 다음과 같다.
```py
['me', 'and', 'you']
```

리스트에서 요소의 순서를 역순으로 돌리고 싶을 때는 `reverse()` 방법을 사용한다. 
```py
team.reverse()
print(team)
```
결과는 다음과 같다.
```py
['you', 'and', 'me']
```

요소를 제거하고 싶을 경우는 `remove()` 방법을 적용한다.
```py
team.remove("and")
print(team)
```
결과는 다음과 같다.
```py
['you', 'me']
```

알파벳이나 숫자 요소를 크기 순서대로 배열하고자 할 때는 `sort()` 방법을 사용한다.
```py
team.sort()
print(team)
```
결과는 다음과 같다.
```py
['me', 'you']
```

리스트에서 특정 요소의 개수를 알고 싶을 때는 `team.count("me")`와 같이 하면 된다.
```py
print(team.count("me"))
```
결과는 다음과 같다.
```py
1
```

위에서 소개한 내용은 방법(methods)에 속하므로 `리스트.방법`의 형식으로 사용해야 함을 유의한다.

### 리스트에서의 함수 사용법
---
리스트에 적용되는 함수 중 자주 사용되는 것은 다음과 같으며 형태는 위에서 언급한 방법(method)과 유사하나 적용 방식이 다름을 유의해야 한다.

| 함수 | 내용 | 예 |
| :--- | :--- | :--- |
| abs() | 절댓값 | abs(-3) |
| int() | 정수로의 변환 | int(9.7) |
| max() | 최댓값 찾기 | max(a) | 
| min() | 최소값 찾기 | min(a) |
| pow() | 승수 계산 2⁴ | pow(2, 4) |
| round() | 반올림 | round(5.821) |

다음은 리스트에 대한 다양한 함수의 사용에 대한 예제 프로그램과 수행결과이다. 절댓값의 결과는 다음과같다.
```py
a = abs(-3)
print(a)
```
결과:
```py
3
```

정수로의 변환은 다음과 같이 한다.
```py
i = int(9.7)
print(i)
```
결과: 
```py
9
```

리스트 `alist = [3, 8, 7, 2]`에서의 최댓값은 다음과 같이 찾을 수 있다.
```py
alist = [3, 8, 7, 2]
mx = max(alist)
print(mx)
```
결과:
```py
8
```

리스트 `alist = [3, 8, 7, 2]`에서의 최소값은 다음과 같이 찾을 수 있다.
```py
alist = [3, 8, 7, 2]
mi = min(alist)
print(mi)
```
결과:
```py
2
```

승수는 다음과 같이 계산된다.
```py
pw = pow(2, 4)
print(pw)
```
결과:
```py
16
```

반올림은 다음과 같다. 
```py
rd = round(5.821)
print(rd)
```

결과:
```py
6
```
