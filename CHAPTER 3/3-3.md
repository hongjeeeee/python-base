# 3.3 투플
투플(tuple)은 리스트와는 달리 포함된 요소를 **바꿀 수 없는** 데이터 형식을 의미한다. 투플은 리스트와 마찬가지로 순서대로 표시된 개체이며 `()` 기호를 써서 표시한다. 다음의 예에서 투플의 정의와 항목을 찾아내는 과정을 여주고 있다.
```py
tuple_1 = ("I", "love", "Python")
tuple_2 = (30, 60, 100, 3000)
print(tuple_1[1])
print(tuple_2[2])
```
이 프로그램을 실행하면 `tuple_1[1]`와 `tuple_2[2]`의 내용이 다음과 같이 나온다. 즉, `tuple_1`에서 두 번째 요소와 `tuple_2`에서 세 번째와 요소를 각각 출력한다.
```py
love
100
```

투플에서는 각 요소의 내용을 변경하는 것이 불가능하다. 이번에는 결과가 `['I', 'love', 'Python', 3000]`와 같이 나올 수 있도록, 프로그램을 작성해 보기로 한다. 다음 프로그램에서는 투플 `tuple_2`의 4번째 요소인 `3000`을 변수 `end`에 저장하고, 그것을 다시 `tuple_1`의 4번째 항목으로 추가하는 작업을 설명하고 있다. 즉, `tuple_1`의 맨 끝에 있는 요소 `10`을 `tuple_2`에 있는 요소 `3000`으로 바꾸기 위한 시도이다. 
```py
tuple_1 = ("I", "love", "Python", 10)
tuple_2 = (30, 60, 100, 3000)
end = tuple_2[3]
tuple_1[3] = end
```
그러나 이 프로그램을 실행시키면 앞서 사용하였던 리스트와 유사한 결과를 기대하였으나, 다음과 같은 오류 메시지가 출력되며, 그 내용은 투플의 내용을 변경시키지 못한다는 내용이다.
```py
TypeError: 'type' object does not support item assignment
```

이번에는 동일한 내용을 리스트 `list_1`, `list_2`를 사용하여 다음과 같이 프로그램을 작성하였다. 
```py
list_1 = ("I", "love", "Python", 10)
list_2 = (30, 60, 100, 3000)
end = list_2[3]
list_1[3] = end
print(list_1)
```
이 프로그램을 실행시키면 다음과 같은 결과를 얻을 수 있으며, 투플과 달리 리스트에서는 요소의 성분을 마꿀 수 있음을 알 수 있다. 
```py
['I', 'love', 'Python', 3000]
```
