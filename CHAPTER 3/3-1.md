# 3.1 리스트
- 리스트(list)는 파이썬에서 가장 많이 사용되는 데이터 관련 도구이다.
- 함수의 반환 값 등에는 리스트와 비슷하지만 특성이 다른 투플(tuple)이 사용된다.
- 리스트와 투플은 다양한 성분을 담을 수 있는 그릇과 같으며 다양한 형식의 자료를 넣을 때 사용되는 편리한 구조이다.

다음 예에서는 `hero`라는 리스트에 문자열 `'Iron Man'`을 할당하는 방법을 설명하고 있다. 
```py
hero = []
hero = ['Iron Man']
print(hero)
```
이 프로그램에서는 `hero = []`로 하여 먼저 비어있는 리스트를 만들고 `hero`에 `'Iron Man'`이라는 문자열을 저장한다. 리스트에서는 `[]` 기호를 사용하여 각 요소를 담는다. 이 프로그램을 실행하면 다음과 같은 결과를 얻는다.
```py
['Iron Man']
```

### 리스트에 요소 추가하기 
---
리스트에는 다음과 같이 다수의 문자열도 저장할 수 있다. 
```py
heros = []
heros = ['Iron Man', 'Thor', 'Black Widow']
print(heros)
```
이 프로그램에서는 리스트를 초기화한 후 `heros`에 다음과 같이 세개의 문자열을 한꺼번에 저장하였다. 이 프로그램을 실행하면 다음과 같은 결과를 얻는다. 
```py
['IronMan', 'Thor', 'Black Widow']
```
`heros` 리스트에 `'Captain America'` 요소를 추가하고 싶은 경우는 `append()`를 사용하여 다음과 같은 방식으로 하면 된다.
```py
heros.append('Captain America')
print(heros)
```
이 프로그램을 수행하면 모두 네개의 요소가 의도한 바와 같이 `heros`에 저장되었음을 확인할 수 있다. 
```py
['Iron Man', 'Thor', 'Black Widow', 'Captain America']
```

### 리스트에 요소 삭제하기 
---
리스트의 요소를 삭제하고자 할 때는 `remove()`를 다음과 같이 사용한다.
```py
heros.romove('Thor')
print(heros)
```
이 프로그램을 실행하면 다음과 같이 `'Thor'`이 리스트 `heros`에서 삭제되었음을 확인할 수 있다.
```py
['Iron Man', 'Black Widow', 'Captain America']
```

### 리스트에 요소 불러오기
---
리스트에 있는 특정한 요소를 불러와서 사용할 수 있다. 현재 리스트 `heros`에 `['Iron Man', 'Black Widow', 'Captain America']`가 저장되어 있다고 할 때, `'Black Widow'`를 불러오기 위해서는 다음과 같이 한다.
```py
print(heros[1])
```
여기서 `heros[1]는 리스트 `heros`에서 두 번째 요소를 불러오는 것을 의미한다. 파이썬에서는 모든 지수를 0으로부터 시작하기 때문에 두 번째 요소를 지칭하기 위해서는 인덱스를 1로 하여야 함을 유의해야 한다. 이 프로그램을 수행하면 다음과 같다.
```py
['Black Widow']
```
