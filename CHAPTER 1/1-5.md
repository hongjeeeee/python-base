# 이름 정하기
프로그램에서는 사용되는 변수, 상수, 함수, 리스트 등에 적절한 이름을 지정해 주어야 한다. 기본적으로 이름은 다음과 같이 영문 소문자와 대문자를 이용하여 정한다.
- x(단일 소문자)
- X(단일 대문자)
- group(소문자)
- GROUP(대문자)

흔히 번거롭다는 이유로 변수 이름을 의미 없이 정하면 나중에 프로그램을 검토할 때 변수의 내용을 이해하기 어려워서 불편할 수 있다. 따라서 이름은 가능하면 구체적으로 정하는 것이 좋으며 이렇게 해두면 나중에 프로그램을 디버깅하기에도 편리하다. 많이 사용되고 있는 작명 형식은 **낙타 형식(camel case)** 과 **뱀 형식(snake case)** 의 두 가지의 종류가 있다.

### 낙타 형식
--- 
이름을 지을 때 관련된 정보를 모두 이어서 붙이는 방식이다. 예를 들어 sky, walker와 return of Jedi는 각각 다음과 같이 표시한다. 
```python
SkyWalker (각 단어의 첫글자를 모두 대문자로)
skyWalker (첫 단어의 첫글자만 소문자로)
returnOfJedi (첫 단어의 첫글자만 소문자로)
```

### 뱀 형식
---
작명할 때 관련된 정보 사이에 `_`을 넣어서 만드는 방식이다. 다음의 예와 같이 사용할 수 있다. 
```python
sky_walker
Sky_Walker
SKY_WALKER
return_of_Jedi
```

### 추천되지 않는 방법
---
다음과 같은 영문자를 이름으로 사용하는 경우는 프로그램을 읽는데 혼동을 주게 될 가능이 있으므로 가능한 사용하지 않는 것이 좋다.
```python
l (소문자 L)
O (대문자 O)
I (대문자 I)
```

이러한 문자를 이름으로 사용하면, 다음 예에서 볼 수 있듯이 왼쪽의 이름과 오른쪽의 숫자가 모양이 유사하여 혼돈을 줄 수 있음을 보여주고 있으며 잘 구분이 되지 않아서 프로그램 오류를 찾고자 할때도 어려울 수 있다.
```python
l = 1
O = 0
I = 1
```

### 상수
---
프로그램 전역(global)에서 사용될 상수는 MAX_NUMBER나 PLANCK_CONSTANT와 같이 대문자로 나타내는 것이 일반적이다. 다음 프로그램에서는 상수 MAX_NUMBER, PLANCK_CONSTANT와 G를 규정하는 방법을 보여주고 있다. 
```python
MAX_NUMBER = 300
PLANCK_CONSTANT = 6.62607004e-34
G = 9.81
```
